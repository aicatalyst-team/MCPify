# Roadmap

- Analyze backend services and frontend actions.
- Detect events, workflows, and permissions.
- Generate runnable MCP server projects.
- Bind generated handlers to exported app functions.
- Provide setup docs for local MCP clients.

- Project config file support.
- More framework-specific analyzers.
- Richer generated tests for server output.
- Hosted documentation and example galleries.
- Deeper security simulation and policy customization.
